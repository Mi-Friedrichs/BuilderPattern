# Demo eines Builder Pattern
Design Pattern 'Builder'

- Komplexe Object Erstellung 
- Erstellungsprozess ist teuer (Performance, Speicher) 
- Sammlung von Daten 

	- Sammeln 
	- Auslösen / Ausführen  
	- Beispiel Konfiguration einer Anwendung (..Builder) 
	- ReservationBuilder
	- ContractBuilder (z.B. als Wizard)
